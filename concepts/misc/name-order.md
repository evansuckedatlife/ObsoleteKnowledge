# Name Order

## summary

*Name order* refers to the sequence in which given names and family names appear in a person's full name, which varies dramatically across cultures. Western convention (given name first, family name last) is not universal, leading to persistent confusion in international contexts, scholarship, and media.

## you gotta know

- Western European and North American tradition places the given name first, family name last (e.g., John Smith); this became dominant through European colonial influence but is not the default globally.
- East Asian cultures—China, Japan, Korea, Vietnam—traditionally place the family name first, then given names (e.g., Confucius's name was Kong Fuzi, where Kong is the family name).
- Hungarian and some Slavic cultures also reverse the Western order—family name first—but this convention is sometimes dropped when speakers adopt Western naming in English-language contexts.
- Many databases and forms assume Western name order, causing errors when processing names from cultures with different conventions; a Vietnamese person named Nguyễn Văn A is often incorrectly indexed under "Văn" or "A" by systems expecting family-name-last.
- Some cultures do not have a concept of "family name" in the Western sense—many Indigenous cultures, some African traditions, and Arab cultures have patronymic or clan systems that function differently.
- The confusion is compounded when scholars transliterate names; classical Chinese and Japanese names often require context to parse correctly (e.g., is "Zhang Wei" a given name or family name?).
