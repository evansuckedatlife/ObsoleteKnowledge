# Exponential Function

## summary

An *exponential function* has the form `f(x) = aˣ` (or more generally `f(x) = abˣ`), where the variable appears as an exponent and the base is a fixed positive number. Exponential functions model explosive growth or decay—population dynamics, radioactive decay, interest compounding, disease spread—and are ubiquitous in the natural and social sciences.

## you gotta know

- Standard form: `f(x) = aˣ` for constant *a* > 0, *a* ≠ 1; or `f(x) = ab^(cx)` for scaling and shifting.
- The natural exponential function uses base *e* ≈ 2.718: `f(x) = eˣ`; this is unique because its derivative is itself: `d/dx eˣ = eˣ`.
- Domain is all real numbers; range is (0, ∞) if *a* > 0. The function is always positive and continuous.
- For *a* > 1, the function grows exponentially; for 0 < *a* < 1, it decays exponentially.
- The graph has a horizontal asymptote at *y* = 0 (for *a* > 1, as *x* → −∞; for 0 < *a* < 1, as *x* → +∞).
- Exponential and logarithmic functions are inverses: if `y = aˣ`, then `x = logₐ(y)`.
