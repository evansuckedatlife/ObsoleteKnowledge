## summary

The **CPU** (central processing unit) is the primary computational engine of a computer, executing program instructions and performing arithmetic and logical operations. It interprets instructions from memory, coordinates with other components, and determines the overall speed and capability of the machine.

## you gotta know

- The *fundamental role* of the CPU is to fetch instructions from memory, decode them, and execute them — the core fetch–decode–execute cycle that drives all computation.
- CPUs are measured by *clock speed* (in gigahertz, GHz), which counts how many instruction cycles occur per second; higher clock speeds generally mean faster computation.
- Modern CPUs have *multiple cores* — independent processing units on a single chip — allowing them to execute different instructions in parallel and tackle multitasking more efficiently.
- The CPU connects to all other components via the *motherboard* and its *bus* architecture, serving as the "conductor" that orchestrates data flow throughout the system.
- *Cache memory* (L1, L2, L3) sits on or near the CPU die, holding frequently-used data for rapid access and dramatically speeding up computation compared to main memory (RAM).
- Power consumption and heat generation scale with clock speed and core count; this is why CPUs require *cooling solutions* (fans, liquid loops) to prevent overheating.
- *Instruction set architecture* (ISA) — such as x86/x64 (Intel/AMD) or ARM — defines the vocabulary of instructions a CPU can understand; software must be compiled for the target ISA.
- Consumer CPUs today feature *integrated graphics* (APUs), eliminating the need for a discrete GPU for basic display output.

## connections

- [[motherboard]] — the CPU plugs directly into the motherboard socket and depends on it for power, cooling, and data routing.
- [[ram]] — the CPU constantly reads instructions and data from RAM and writes results back; memory bandwidth is a critical bottleneck.
- [[graphics-card]] — the CPU shares the workload with a discrete GPU for heavy graphics or compute tasks.
- [[power-supply-unit]] — supplies the CPU with stable, adequate power; insufficient power delivery causes throttling or crashes.
- [[cooling]] — manages the heat generated by the CPU's electrical switching; failure to cool causes thermal shutdown.
